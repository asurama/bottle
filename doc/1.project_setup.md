# 1. 초기 프로젝트 세팅 및 뼈대 구성 (Next.js)

## 1. 왜 이런 작업이 필요한가?
'보틀' 웹사이트의 클라이언트 및 서버 API를 통합하여 효율적으로 개발하기 위해, 가장 최신 프레임워크인 Next.js 기반 환경을 구축해야 합니다. 또한 고급스러운 UI와 모바일 반응형 카드 컴포넌트를 쉽고 일관성 있게 구현하기 위해 `shadcn/ui` 및 Tailwind CSS의 뼈대를 선제적으로 마련해야 합니다.

## 2. 어떻게 구현하는가?
*   로컬 디렉토리(`D:\work\개인\code\bottle`)에 `create-next-app` 명령어를 사용하여 TypeScript, Tailwind CSS, App Router가 적용된 프로젝트를 생성합니다.
*   명령어: `npx -y create-next-app@latest . --typescript --tailwind --eslint --app --src-dir --import-alias "@/*"`
*   shadcn/ui 초기화 후 기본 테마(다크 모드 기준) 색상을 `globals.css`와 `tailwind.config.ts`에 세팅합니다.
